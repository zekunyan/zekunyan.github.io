    CentOS 6.5 RCPR web环境配置

    基本要求

    组件总览：

    1. armabillo，opencv 2.4.x，git
    2. g++、cmake、automake等。
    3. apache，php
    4. thrift

    安装基本组件：

    sudo yum -y groupinstall "Development Tools"

    安装Thrift

    升级 autoconf

    wget http://ftp.gnu.org/gnu/autoconf/autoconf-2.69.tar.gz tar xvf autoconf-2.69.tar.gz cd autoconf-2.69 ./configure --prefix=/usr make sudo make install cd ..

    升级 automake

    wget http://ftp.gnu.org/gnu/automake/automake-1.14.tar.gz tar xvf automake-1.14.tar.gz cd automake-1.14 ./configure --prefix=/usr make sudo make install cd ..

    升级bison

    wget http://ftp.gnu.org/gnu/bison/bison-2.5.1.tar.gz tar xvf bison-2.5.1.tar.gz cd bison-2.5.1 ./configure --prefix=/usr make sudo make install cd ..

    安装 C++ Lib Dependencies

    sudo yum -y install libevent-devel zlib-devel openssl-devel

    安装Boost

    wget http://sourceforge.net/projects/boost/files/boost/1.55.0/boost_1_55_0.tar.gz tar xvf boost_1_55_0.tar.gz cd boost_1_55_0 ./bootstrap.sh sudo ./b2 install

    安装 Thrift

    git clone https://git-wip-us.apache.org/repos/asf/thrift.git cd thrift ./bootstrap.sh ./configure --with-lua=no make sudo make install

    安装 OpenCV

    wget http://jaist.dl.sourceforge.net/project/opencvlibrary/opencv-unix/2.4.8/opencv-2.4.8.zip unzip opencv-2.4.8.zip mv opencv-2.4.8 path_of_your_workspace #以后还需要 cd path_of_your_workspace/opencv-2.4.8/ mkdir release cd release cmake -D CMAKE_BUILD_TYPE=RELEASE -D CMAKE_INSTALL_PREFIX=/usr/local .. make sudo make install

    安装 Armabillo

    wget http://sourceforge.net/projects/arma/files/armadillo-4.650.4.tar.gz tar zxf armadillo-4.650.4.tar.gz cd armadillo-4.650.4 ./configure make sudo make install

    安装Apache、PHP

    Apache

    • yum -y install httpd
    • chkconfig --levels 235 httpd on #开机启动
    • /etc/init.d/httpd start #开启服务

    目录：

    • /var/www/html : web 根目录。
    • 配置文件：/etc/httpd/conf/httpd.conf
    • 其余配置文件：/etc/httpd/conf.d/

    PHP

    • yum -y install php
    • /etc/init.d/httpd restart # 重启apache

    安装PHP扩展

    • yum -y install php-gd php-imap php-ldap php-odbc php-pear php-xml php-xmlrpc php-mbstring php-mcrypt php-mssql php-snmp php-soap php-tidy curl curl-devel
    • yum -y install php-pecl-apc
    • /etc/init.d/httpd restart # 重启apache

    配置

    apache

    编辑配置文件：/etc/httpd/conf/httpd.conf。 修改web根目录的为“AllowOverride All”。 重启。

    /var/www/html

    创建“upload”目录，设置777权限。（因为web程序以apache账户运行）

    安装编译RCPR C++后端程序

    下载、编译

    cd 某个目录 git clone git@git.coding.net:zekunyan/RCPR_Thrift.git #120.24.247.190的ssh公钥已经添加到coding上。 cd RCPR_Thrift/RCPR/server mkdir build cd build cmake .. # 采用外部编译，两个点 make

    运行

    • 将model.bin、haarcascade_frontalface_alt.xml复制到本目录（build目录）
    • 切换到root，（sudo -i，避免文件读写错误）
    • nohup ./RCPR_Server.run & （后台运行）
    • 可以通过查看nohup.out文件看log。

    停止程序

    • ps aux | grep RCPR
    • kill -9 (对应的进程PID)

    安装RCPR PHP端程序

    cd /var/www/html git clone git@git.coding.net:zekunyan/RCPR_PHP.git mv RCPR_PHP RCPR

    测试：http://120.24.247.190/RCPR/test/index.html

    编写清理上传图片临时目录脚本

    因为测试页面的图片要显示，不能马上删除，所以要编写脚本，定时删除。

    脚本内容：

    #!/bin/sh

echo 'Begin clean at' `date`
cd /var/www/html/upload
echo 'Total files: ' `ls -l | wc -l`
rm -rf *
echo 'Clean done at ' `date`

    设置crontab，每天23点删除：

    0 23 * * * sh /home/yzk/program/crontab/cleanUploadImg.sh >> /home/yzk/program/crontab/cleanLog.txt 2>&1

    能看到这的都是真爱，打个赏呗=。=
